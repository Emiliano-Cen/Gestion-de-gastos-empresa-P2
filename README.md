Aplicación de Gestión – Programación 2
- Aplicación desarrollada como proyecto académico en el curso de Programación 2, enfocada en la implementación de lógica más avanzada, organización del código y resolución de problemas complejos.
- El sistema permite gestionar información y aplicar reglas de negocio mediante una estructura de código más modular y mantenible.

---

Tecnologías utilizadas
- C#
- Programación orientada a objetos (POO)
- Manejo de estructuras de datos
- Consola / Aplicación de escritorio (según corresponda)

---

Funcionalidades principales
- Gestión de entidades del sistema
- Implementación de lógica de negocio
- Validaciones de datos
- Manejo de estados dentro de la aplicación
- Interacción con el usuario mediante consola/interfaz

---

Conceptos aplicados
- Programación orientada a objetos:
  - Clases y objetos
  - Encapsulamiento
  - Métodos y propiedades
- Modularización del código
- Manejo de estructuras de datos
- Validación y control de errores
- Separación de responsabilidades

---

Objetivo del proyecto
Aplicar conceptos intermedios de programación para desarrollar una aplicación funcional, organizada y escalable, mejorando la calidad del código respecto a proyectos iniciales.

---

Cómo ejecutar

1. Clonar o descargar el repositorio
2. Abrir el proyecto en Visual Studio
3. Ejecutar la aplicación
